多节点惯性姿态采集系统：从传感器节点到人体运动捕捉 Multi-Node Inertial Motion Capture System

本文梳理多节点人体动作捕捉系统的整体架构，包括惯性传感器数据采集、零偏校准、姿态解算、无线通信、上位机接收和人体姿态重构等关键环节。 This article introduces the architecture of a multi-node inertial motion capture system.

一、系统整体架构 1. System Architecture

系统由多个可穿戴惯性传感器节点、无线通信网络、数据接收终端和人体姿态重构程序组成。每个节点负责采集对应人体肢段的加速度、角速度和磁场信息，并通过姿态融合算法计算四元数。 The system consists of multiple wearable sensor nodes, wireless communication, a receiver and motion reconstruction software.

二、数据处理流程 2. Data Processing Flow

  • 读取加速度计、陀螺仪和磁力计原始数据； Acquire accelerometer, gyroscope and magnetometer data.
  • 完成零偏补偿、尺度校准与异常值检查； Perform bias correction and calibration.
  • 使用互补滤波或扩展卡尔曼滤波完成姿态融合； Estimate attitude using complementary filtering or EKF.
  • 将节点编号、时间戳和姿态四元数组帧发送。 Transmit node ID, timestamp and quaternion data.

三、主要工程难点 3. Engineering Challenges

实际开发中需要重点解决节点时间同步、陀螺仪漂移、磁场干扰、无线丢包和传感器安装误差。相比单节点姿态解算，多节点系统更强调各节点之间的时间同步和坐标系一致性。 Major challenges include synchronization, sensor drift, magnetic interference, packet loss and sensor alignment.

运动状态下的PPG信号容易受到手腕位移、皮肤接触变化和环境光干扰。本文介绍如何利用三轴加速度信息识别运动伪影，并提高心率输出的连续性和稳定性。 This article discusses how accelerometer information can improve motion PPG heart-rate estimation.

一、运动伪影来源 1. Motion Artifacts

PPG通过检测血容量随心搏产生的周期性变化估计心率。在运动状态下，传感器与皮肤之间的相对位移会引入明显的运动伪影，其频率可能与真实心率频率接近，导致频谱峰选择错误。 Motion artifacts may overlap with the true heart-rate frequency and cause incorrect spectral peak selection.

二、算法处理流程 2. Processing Pipeline

  • 对PPG信号进行去趋势、滤波和归一化； Detrend, filter and normalize the PPG signal.
  • 采用8秒窗口、2秒步长进行分段处理； Use an 8-second window with a 2-second step.
  • 提取频谱候选峰并结合历史心率进行连续跟踪； Track spectral candidates using previous estimates.
  • 使用ACC对剧烈运动状态进行异常拒绝和质量门控。 Use ACC as a quality gate during severe motion.

三、嵌入式部署思路 3. Embedded Deployment

对资源有限的嵌入式设备而言，复杂的自适应修正策略并不一定优于简单方法。采用固定跟踪、变化限幅和极端异常门控，可以降低运算量并提升输出稳定性。 Fixed tracking, rate limiting and motion gating can provide stable results with lower computational cost.

一、设计目标 1. Design Objective

机构需要在较少驱动源的条件下产生上下、左右和前后方向的耦合运动，使座椅输出类似马术治疗或步行骨盆运动的空间轨迹，同时保证运动幅度可以调节。 The mechanism aims to reproduce coupled pelvic motion using a compact single-motor structure.

二、机械结构组成 2. Mechanical Structure

  • 可调偏心套筒控制水平运动幅度； An adjustable eccentric sleeve controls amplitude.
  • 圆柱凸轮产生周期性的升降运动； A cylindrical cam generates vertical motion.
  • 棘轮机构实现单向锁紧和反向调节； A ratchet enables locking and adjustment.
  • 多个运动分量耦合形成空间“V8”形轨迹。 Coupled motions form a spatial figure-eight path.

三、仿真与验证 3. Simulation and Verification

使用MATLAB建立运动学模型并优化关键尺寸参数，在SolidWorks中完成三维建模、装配和机构干涉检查，最后通过座椅参考点轨迹验证机构输出效果。 MATLAB was used for kinematic optimization, while SolidWorks was used for modeling, assembly and interference checking.

一、先验证再制板 1. Prototype Before PCB

在绘制PCB之前，先使用开发板和面包板验证传感器通信、供电电压、程序接口和无线模块，可以提前排除器件选型和通信协议层面的问题。 Breadboard validation helps identify communication and power issues before PCB fabrication.

二、PCB设计重点 2. PCB Design Priorities

  • 电源入口设置保护、滤波和稳压电路； Add protection, filtering and regulation.
  • 去耦电容靠近芯片电源引脚放置； Place decoupling capacitors near power pins.
  • 保留SWD下载接口和关键测试点； Keep SWD and important test points.
  • 数字通信线尽量缩短并保持连续参考地。 Keep digital traces short with continuous ground.

三、推荐调试顺序 3. Debugging Sequence

调试时先断电检查短路，再测量各级电源，随后下载最小程序，最后依次测试传感器、显示模块和无线通信。分模块调试可以避免多个问题同时出现时难以定位故障。 Check shorts first, verify power rails, run minimal firmware and then test each module.
